# llm-benchmark
This repository benchmarks the performance of open-source language models on local machines.

## Models
* LLaMA 3.1 8B
* Qwen 2.5
* Gemma 3 4B

## Setup
```bash
python3 -m venv .env
source .env/bin/activate
pip install -r requirements.txt
python benchmark/main.py
```
